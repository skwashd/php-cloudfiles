				Autoload Rackspace CloudFiles classes

				

				
Instead of filling your code with require_once statements you
				can use the following code to "automagically" include
				classes as they are needed.

				

				
    
					
  1. 
					
     require_once '/path/to/Rackspace/CloudFiles.php';
    
					
    2. 
					
  2. 
					
     spl_autoload_register(array('Rackspace_CloudFiles', 'autoload'));
    
					
    3. 
				

				

				
This isn't needed if you are already using a PEAR
				(compatiable) autoloader.
